Cyberpunk Calculator
A futuristic, cyberpunk-themed calculator built using HTML, CSS, and JavaScript. This project demonstrates a modular approach by separating the code into individual files for structure, styling, and functionality.

Overview
The Cyberpunk Calculator is a visually striking calculator that combines a dark, neon-lit cyberpunk aesthetic with essential arithmetic functionality. The project is developed in English and uses modern web development practices to ensure clean separation between HTML, CSS, and JavaScript.

Features
Basic Arithmetic Operations: Addition, subtraction, multiplication, and division.

Advanced Functionality:

Parentheses Support: Allows the user to group operations.

Backspace: Removes the last entered character.

Plus/Minus Toggle: Changes the sign of the current number.

Cyberpunk Themed UI:

Neon green accents with dark backgrounds.

Glow effects and modern typography.

Responsive Design: The layout adapts to various screen sizes for a seamless user experience.
